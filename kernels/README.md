# 📂 kernels/

Каталог с исходниками ядер.

## Пример структуры

```
kernels/
├── linux-5.10/
├── linux-5.10-patched/
├── linux-6.1/
├── linux-6.1-patched/
├── linux-6.12/
```


## Правила
- `linux-<версия>/` — vanilla ядро.
- `linux-<версия>-patched/` — ядро с патчами (CentOS, Elbrus).
- RT-патчи применяются внутри vanilla-каталогов, отдельной папки не нужно.
